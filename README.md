\# Tool by Yokus



Многофункциональный инструмент для оптимизации Windows и установки программ.



\## Возможности



\### 🧹 Очистка

\- Очистка Temp

\- Очистка DXCache (NVIDIA)

\- Очистка OpenGL Cache

\- Очистка Prefetch

\- Очистка DNS кэша

\- Очистка RAM

\- Очистка корзины



\### 📦 Софт

\- Браузеры (Chrome, Opera, Yandex, Firefox)

\- Платформы для игр (Steam, Epic, Rockstar, Origin, Battle.net)

\- Общение (TeamSpeak, Discord, Telegram)

\- Другое (WinRAR, 7-Zip, VLC, qBittorrent, VC++, .NET)



\### ⚡ MUST HAVE

Автоматическая установка базового набора программ одним кликом.



\## Установка



\### Для пользователей (EXE)

1\. Скачайте последнюю версию из \[Releases](https://github.com/Yokus-pyth/Windows-tool-after-reinstall-OS/releases)

2\. Запустите `tool\_by\_yokus.exe`



\### Для разработчиков

```bash

pip install -r requirements.txt

python tool\_by\_yokus.py
